    • How to use easyModal

    EasyModal is a little library made by weasywig to create a better interaction with the user. The aim of this library is to easily create modals to ask questions to the user, such as a simple text, a confirmation.
    For each types, differents options can be given, and the function return differents results. In this documentation, you will find what options you can give for each types, and what values are returned.

    An example of simple modal

    EasyModal types

    Here is the list of the existing types of modals that easyModal offers :

    Alert Confirm Prompt Text and textarea Text and select Custom

    Whatever the type of modal you choose, the callback function will always have two arguments :
    modal the DOM object of the modal
    event an object containing the submit event, and an object containing the hide event

    Alert

    The aim of this modal is just to aware the user of something, the user cannot give informations with this modal.

    Type name : easyModal.types.ALERT

    Usage

    easyModal.alert("This is an alert",function(modal,event){
    console.log("end alert");
});

    Return values

    None, except the two default values modal and event

    Result

    Confirm

    This modal is used to ask a user to confirm an action (Like the window.confirm()) with the difference that calling the easyModal.cnofirm will not stop the script from running.

    Type name : easyModal.types.CONFIRM

    Usage

    //Usage
easyModal.confirm("Are your sure ?",function(modal,event,confirmed){
    confirmed ? console.log("So you're sure") : console.log("ok let's cancel'");
});
//or
easyModal.confirm({
    title : "Are you sure ?", //The main title of the modal
    text : "Do you really want to delete this ?" //The text to display in the modal body
},function(modal,event,confirmed){
    confirmed ? console.log("Ok, lets delete it") : console.log("So we dont delete");
});

    Return values

    confirmed : this third argument is a boolean. Which is true if the user clicked 'ok' and false if the user clicked 'cancel'

    Result

    Prompt

    This modal contain a single input of text type. To ask the user a single string. Its the same as window.prompt() with the difference that the script is not stopped.

    Type name : easyModal.types.PROMPT

    Usage

    //Easy way
easyModal.prompt("How old are you ?", function (modal, event, age) {
    console.log("So you tell me you're'" + age);
});
//or custom way
easyModal.prompt({
    'title' : 'How old are you ?', //title of the input
    'input' : 'number', //The type of the input
    'placeholder' : 20,
    'min' : 0, //The min value the user can submit
    'max' : 124 //The max value the user can submit
}, function (modal, event, age) {
    console.log("So you tell me you're'" + age);
});

    Return values

    age : The third argument of the function is a string by default. If you chose a input of type 'number', a number instead of a string will be returned.

    Result

    Select

    This modal contain a single input. It's a select. The user need to select a value in those proposed

    Type name : easyModal.types.SELECT

    Usage

    easyModal.select({
    'title': 'What is your favorite fruit', //Modal title
    'values': ['Apple', 'Mango', 'Pear', 'Strawberry'], //Possible values of the select
    'selectLabel': 'Choose a fruit' //The title of the select
}, function (modal, event, index, value) {
    console.log("You choose the index number " + index, "That's the fruit called '" + value);
});

    Return values

    index : the index of the selected option

    value : the value of the selected option

    Result

    Text and Textarea

    This modal contain two inputs : a simple text input, and a textarea.

    Type name : easyModal.types.TEXTNAREA

    Usage

    //Basic use :
easyModal.textNArea("Name and description", function (modal, event, text, area) {
    console.log("Your name is ", text);
    console.log("Your description is ", area);
});
//custom use
 easyModal.textNArea({
    'title' : 'Name and description',
    'textLabel' : 'Your name',
    'areaLabel' : 'Your description'
 },function(modal,event,tex,area){
     console.log("Your name : " + tex);
     console.log("Your description : " + area);
 });

    Return values

    text : the string the user type in the text input

    area : the string the user type in the textarea

    Result

    Text and Select

    This modal looks like the text and area one. But instead of a textarea, there is a select input, where the user have to choose a value.

    Type name : easyModal.types.TEXTNDROPDOWN

    Usage

    easyModal.textNDropDown({
    'title' : 'Whats your favorite number and color ?', //Modal title
    'input' : 'number', // The type of the first input
    'selectVal' : ['Red','Blue','White','Black','Gray','Green'], //The values of the select
    'selectLabel' : 'Pick a color', //The label of the select
    'textLabel' : '9' //The placeholder of the text
},function(modal,event,text,index,selValue){
    console.log("Your favorite number is " + index);
    console.log("And you selected the index number " + index + " as your favorite color");
    console.log("Which correspond to the value '" + selValue + "'. Great !");
});

    Return values

    text : the string the user typed into the first input

    index : the number correponding to the index of the selected option

    selValue : the string corresponding to the value of the selected option

    Result

    Custom

    Here, you can put whatever you want as title, body and buttons.

    Type name : easyModal.types.CUSTOM

    Usage

    easyModal.custom({
        'title': 'The end of the API',//Modal title
        //The content, it can be HTML as well as a simple string
        'content': 'Here is the end of the API documentation <br/> <img src="assets/images/api.png" /><br/> Hope it helped you',
        'buttons': easyModal.templates.submitButton,//The bottom-buttons
        'submitCallback': function (modal, event) {
            console.log("You submitted this modal");
            console.log("Thank you for you attention");
        },//The function to call when the user close the modal
        'showCallback': function (modal, event) {
            console.log("Just came to show the modal : ");
            console.log(modal);
        },//A function to call when the modal is shown
        'size': 'modal-lg'//The modals size
    });

    Return values

    None, except as usual, the modal and the event

    Result
