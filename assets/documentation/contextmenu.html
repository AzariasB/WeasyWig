    The contextmenu is the list that is displayed when the user right-click on a created div. Here is what it looks like

    As you can see, differents options are displayed. Lets see how to add your own options to this contextmenu

    Name your action

    First, you have to give a name to your contextmenu, here's how it's done :

    {
    "name": "Edit text",
    "action": function (target, view) {...}
},

    The value you give for the key 'name' will be the text displayed in the contextmenu

    The called function

    When the user will click on an option in the contextmenu, a function is called. This function takes in parameter the target (where the user right-clicked) and the view. The function does not need to return anything. 

    {
    "name": "...",
    "action": function (target, view) {
        var options = {
            'title': "Change text",
            'size': 'modal-lg',
            'placehloder ': $(target).text()
        }

        view.showModal(easyModal.types.PROMPT, options, function (modal, event, res) {
            $(target).text(res);
        });
    }
},

    In this simple example, the function change the text of the target. To do that, a prompt is shown to the user to ask him the new text, and the result is applied to the div where the user right-clicked.

    Contextmenus subemenu

    If you want to change only a certain style, but you don't want to have the contextmenu all over the page, you can create submenu, as follow. 

    "name": "Change border",
            "submenu": [
                {
                    "name": "Color",
                    "action": function (target) {...}
                },

    The submenu takes in parameter an array, containing differents options, as before, with a name and and action funciton. You can also create submenu into submenus (but don't do that too much, it's not good for the user experience)
