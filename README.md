# WeasyWig
A wysiwig editor made for bootstrap creators


## Description
This is a university project.
I had to create a web app. And decided to create a Wysiwyg editor.

## Dependencies : 
 * bootstrap
 * jquery
 * jquery-ui
 * moustetrap
 * underscorejs
